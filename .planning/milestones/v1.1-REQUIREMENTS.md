# Requirements: CHA Bio Complex Fire Safety System

**Defined:** 2026-03-31
**Core Value:** 현장에서 모바일로 소방시설 점검을 기록하고, 법적 요구사항에 맞는 점검일지를 즉시 출력할 수 있어야 한다

## v1.1 Requirements

v1.1 UI 재편 + 기능 확장 마일스톤 요구사항. 각 항목은 로드맵 Phase에 매핑된다.

### 네비게이션

- [x] **NAV-01**: BottomNav에서 더보기를 제거하고 조치 메뉴를 신규 추가, 승강기를 더보기 자리로 이동
- [x] **NAV-02**: SideMenu(햄버거)에 더보기 항목을 통합하고 용도별로 재정리
- [x] **NAV-03**: MorePage(/more) 제거 및 라우트 정리

### 조치 관리

- [x] **REM-01**: 불량/주의 판정된 점검 개소를 목록으로 조회할 수 있다
- [x] **REM-02**: 각 항목에 조치 메모를 작성하고 상태를 변경할 수 있다
- [x] **REM-03**: 조치 전/후 사진을 첨부할 수 있다 (R2)
- [x] **REM-04**: 카테고리/상태/기간별로 필터링/검색할 수 있다

### 식사 기록

- [x] **MEAL-01**: 개인별 조식/중식/석식 식사 여부를 기록할 수 있다
- [x] **MEAL-02**: 월별 식사 통계(횟수, 금액)를 조회할 수 있다
- [x] **MEAL-03**: 주간 식당 메뉴표를 PDF 업로드 또는 수동 등록으로 관리하고, 팀원이 요일별 메뉴를 조회할 수 있다
- [x] **MEAL-04**: 메뉴표 PDF 업로드 시 월~금 점심/저녁 + 토요일 점심 메뉴를 텍스트로 추출해 제공한다

### 보수교육

- [x] **EDU-01**: 교육 일정을 등록/수정/조회할 수 있다 (교육명, 날짜, 기관, 대상자)
- [x] **EDU-02**: 이수 완료를 기록하고 인증서를 업로드할 수 있다 (R2)
- [x] **EDU-03**: 개인 페이지에서 다음 교육 마감일 D-day를 확인할 수 있다

### 관리자 설정

- [x] **ADMIN-01**: 직원 계정을 추가/수정/비밀번호 초기화할 수 있다
- [x] **ADMIN-02**: 시스템 설정(점검 카테고리, 개소 관리 등)을 구성할 수 있다
- [x] **ADMIN-03**: 햄버거 메뉴 항목 순서/표시를 설정할 수 있다

### 기술 개선

- [x] **TECH-01**: 점검자 이름을 DB에서 동적으로 로딩한다 (하드코딩 제거)
- [x] **TECH-02**: 대시보드에 연속 달성일(streakDays)을 계산/표시한다

### 법적 점검

- [x] **LEGAL-01**: 법적 점검 결과를 기록하고 서류를 업로드할 수 있다 (R2)
- [x] **LEGAL-02**: 지적사항을 등록하고 시정조치 기록/완료 확인할 수 있다

### 승강기 검사

- [x] **ELEV-02**: 승강기 법정검사 인증서/리포트를 업로드하고 조회할 수 있다 (R2)

## v2 Requirements

추후 마일스톤으로 이관. 현재 로드맵에 미포함.

### 도면

- **FLOOR-01**: DWG → SVG 변환 후 층별 도면을 앱에서 열람할 수 있다
- **FLOOR-02**: 도면 위에 소방시설 위치를 인터랙티브하게 표시할 수 있다

### 법적 점검 일정

- **LEGAL-03**: 작동기능/종합정밀 점검 연간 일정을 관리하고 D-day 카운트다운을 볼 수 있다

### 승강기 검사 일정

- **ELEV-03**: 승강기별 연간 검사 일정을 관리하고 D-day 카운트다운을 볼 수 있다

## Out of Scope

| Feature | Reason |
|---------|--------|
| AI 보고서 자동작성 | 추후 별도 마일스톤 |
| 모바일 오프라인 점검 | 현재 네트워크 환경 충분 |
| OAuth/소셜 로그인 | 4인 내부 팀 전용 |
| iOS PWA BottomNav 바닥 이슈 | safe-area-inset 관련, 건드리지 말 것 |
| 다중 건물/멀티테넌트 | 단일 건물 전용 |
| Web Push 알림 | 4인 팀에 과도, Workers 호환 이슈 |
| 도면 리뉴얼 (DWG→SVG) | DWG 레이어 미분리로 용량 이슈, v2로 이관 |
