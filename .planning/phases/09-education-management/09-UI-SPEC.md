## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Gap between icon and inline label, badge internal padding |
| sm | 8px | Gap between stacked small elements, button padding-inline |
| md | 16px | Card padding, section padding, header padding |
| lg | 24px | List vertical gap, BottomSheet internal padding |
| xl | 32px | Major section break inside BottomSheet |
| 2xl | 48px | Page top/bottom breathing room |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Touch target minimum: 48px height for all tappable elements (header buttons, card rows, tab bar items) — standardised to match header bar height, all multiples of 4
- Header bar: 48px height fixed — source: MealPage.tsx header pattern (D-02)
- BottomSheet drag handle: 4px tall × 32px wide centered at top — source: AdminPage.tsx BottomSheet

## Color

| Role | Token | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--bg)` `#0d1117` | Page background, list scroll area |
| Secondary (30%) | `var(--bg2)` `#161b22` | Staff education cards, sticky header, BottomSheet |
| Accent (10%) | `var(--acl)` `#3b82f6` | Active tab underline; primary action button background (BottomSheet only) |
| Warning semantic | `var(--warn)` `#f59e0b` | D-day badge when D ≤ 30 and D ≥ 0 |
| Danger semantic | `var(--danger)` `#ef4444` | D-day badge when past deadline (D < 0) |
| Safe semantic | `var(--safe)` `#22c55e` | D-day badge when D > 30 (no urgency) |
| Inset surface | `var(--bg3)` `#1c2128` | Date input field background in BottomSheet |

Accent (`var(--acl)`) is reserved for:
- Completion record type selector active indicator (initial/refresher underline)
- Primary action button background in BottomSheet (이수일 기록 / 수정 완료)
- No other usage in this phase — do not apply to card backgrounds or text

D-day badge background: use 20% opacity of the semantic color as background fill.
Example: warning badge = `background: 'rgba(245,158,11,0.15)'`, text = `color: 'var(--warn)'`

**Focal point:** The D-day badge is the primary visual anchor per card. It must be the highest-contrast element in each row — sized, colored, and positioned (top-right of card) to draw the eye before staff name or other metadata.

## Component Specifications

### StaffEducationCard

Each card represents one staff member and their education status.

```
┌──────────────────────────────────────────────────────────┐
│  [Avatar initial 32×32 circle bg3 t2]                    │
│  김철수 (16px 700 t1)          [D-day badge]             │
│  소방안전관리자 (13px 400 t2)                             │
│  마지막 이수: 2024-03-15 (보수) (12px 400 t3)            │
│  다음 마감: 2026-03-15 (12px 400 t2)                     │
└──────────────────────────────────────────────────────────┘
```

- Background: `var(--bg2)`, borderRadius: 12, padding: 16
- Border: `1px solid var(--bd)`
- Tap entire card → opens BottomSheet for that staff member
- Tap target: full card is tappable, min-height 80px

**Avatar:** 32px circle, background `var(--bg3)`, text `var(--t2)`, font 14px 700. First character of staff name.

**D-day badge (primary visual anchor — highest-contrast element per card):**
- D > 30: background `rgba(34,197,94,0.12)`, text `var(--safe)`, label "D-{N}"
- 0 ≤ D ≤ 30: background `rgba(245,158,11,0.15)`, text `var(--warn)`, label "D-{N}"
- D < 0: background `rgba(239,68,68,0.15)`, text `var(--danger)`, label "D+{|N|} 초과"
- No records + no appointed_at: no badge, show "선임일 미등록" in `var(--t3)` 12px

**교육 유형 label:** "첫 실무교육" (initial, never completed) or "보수교육" (refresher after first)
shown next to the deadline in 12px `var(--t3)`.

**Skeleton state:** `background: var(--bg3)`, borderRadius 12, height 88, `animation: blink 2s ease-in-out infinite` — matches SKELETON_STYLE pattern from RemediationPage.tsx

### BottomSheet (이수 기록 등록/수정)

Opens when user taps a StaffEducationCard. Replicates AdminPage.tsx BottomSheet pattern exactly.

```
[Overlay: rgba(0,0,0,0.6) fixed inset-0]
  └── [Sheet: bg2, borderRadius 16 16 0 0, slideUp animation]
        [Drag handle: 32×4 bd2 centered, paddingTop 12]
        [Title: 16px 700 t1, padding 12 16 0]
        [Staff name subtitle: 13px 400 t2, padding 4 16 8]
        ─────────────────────────────
        [이수일] date picker field
        [교육 유형] select (initial / refresher) — default auto-set
        ─────────────────────────────
        [이수일 기록] primary action button
        [기록 취소] secondary action button
        [bottom safe area padding]
```

**Date input field:** full-width, background `var(--bg3)`, borderRadius 8, padding 12, border `1px solid var(--bd2)`, color `var(--t1)`, fontSize 14. Input type="date".

**교육 유형 select:** same visual as date input. Options: "실무교육 (최초)" / "보수교육". Auto-selects based on existing record count (0 records → 실무교육, ≥1 record → 보수교육). User can override.

**Primary button (이수일 기록):** full-width, height 48, background `var(--acl)`, borderRadius 10, color white, fontWeight 700, fontSize 15.

**Secondary button (기록 취소):** full-width, height 48, background transparent, border `1px solid var(--bd2)`, color `var(--t2)`, fontSize 14. Placed below primary with 8px gap.

**수정 mode:** When editing existing record, title changes to "이수일 수정", primary button label changes to "수정 완료". Education type selector is disabled (read-only, `var(--t3)`, no pointer).

### 선임일 미등록 state (card)

When `staff.appointed_at` is null:
- No D-day badge
- Show inline text "선임일 미등록" — 12px, `var(--t3)`, replacing the deadline row
- Card remains tappable (to allow recording completion once appointed_at is set via Admin)

## Interaction Contract

### Card tap
- Tap anywhere on StaffEducationCard → open BottomSheet for that staff member
- Show records count in subtitle if records > 0: e.g. "이수 이력 2건"
- Tap outside BottomSheet overlay → close without saving

### BottomSheet open
- Only open if current user is admin OR the card belongs to the current user (D-10)
- If neither: do not show any CTA. Card is read-only for other users (no button tap affordance)
- Animation: `slideUp 0.28s ease-out both` (defined in index.css)

### 교육 유형 auto-set
- On BottomSheet open: query existing records count for this staff member
- If count === 0: default type = 'initial' (실무교육), field locked from changing (greyed out)
- If count ≥ 1: default type = 'refresher' (보수교육), field editable

### D-day recalculation
- Recalculate on every render of the card using `calcNextDeadline()` (client-side, date-fns)
- No server-side D-day. API returns raw `appointed_at` + records array only.
- If `appointed_at` is null: skip calculation entirely, render 선임일 미등록

### Navigation
- Header back button (←): `navigate(-1)` — source: MealPage.tsx pattern (D-02)
- /education is in NO_NAV_PATHS (already present in App.tsx line 52) — no BottomNav rendered
