## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, icon button padding |
| md | 16px | Default element spacing, section padding |
| lg | 24px | Section breaks |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- BottomNav total height: `calc(54px + var(--sab, 34px))` — fixed by existing implementation, do not change
- BottomNav tab padding: `3px 0` — preserves existing touch target sizing
- QR button: `margin-top: -14px` — existing floating treatment, preserve as-is
- SideMenu item padding: `9px 13px` — existing pattern, preserve
- Safe area inset: `var(--sat)` (top), `var(--sab)` (bottom) — always include in height calculations
- GlobalHeader height: `48px` — 8-point aligned, matches ElevatorPage header pattern (`padding: 8px 12px`)
- Hamburger / icon button touch target: `32px × 32px` minimum (existing `iconBtnSt` pattern in ElevatorPage)

**Source:** `src/components/BottomNav.tsx`, `src/pages/ElevatorPage.tsx`

## Interaction Contracts

### BottomNav active state
- Active tab: icon color `var(--acl)`, label color `var(--acl)`
- Inactive tab: icon color `var(--t3)`, label color `var(--t3)`
- QR tab: always uses gradient button treatment (never colored `var(--acl)`)
- Transition: none (instantaneous, matching existing implementation)

### GlobalHeader hamburger button
- Press: sets `sideOpen = true` in Layout state
- Button style: `width: 32px, height: 32px, borderRadius: 7px, background: var(--bg3), border: none, color: var(--t2), cursor: pointer, display: flex, alignItems: center, justifyContent: center`
- Icon: 3-line hamburger SVG, `width={15} height={15}`, `strokeWidth={2}`, `stroke="var(--t2)"`

### SideMenu open/close
- Open: `transform: translateX(0)`, transition `0.3s cubic-bezier(.4,0,.2,1)`
- Close: `transform: translateX(-100%)`
- Overlay: `opacity: 0.65 (rgba(0,0,0,0.65))` when open, `0` when closed, `transition: opacity 0.28s`
- Close triggers: overlay click, ✕ button click, item navigation
- zIndex: overlay `190`, panel `200`

### /more redirect
- Behavior: `<Navigate to="/dashboard" replace />` — no animation, no toast
- Seamless — user arriving at /more from a bookmark lands on dashboard without error

### 조치 tab placeholder page
- No interaction except GlobalHeader hamburger (opens SideMenu)
- No buttons, no forms, no loading states
- Static content only
