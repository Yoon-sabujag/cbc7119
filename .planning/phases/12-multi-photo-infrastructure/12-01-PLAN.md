1. Create `cha-bio-safety/src/hooks/useMultiPhotoUpload.ts` — a named export `useMultiPhotoUpload`.

Interface (per RESEARCH.md Pattern 2):
```typescript
const MAX_PHOTOS = 5

interface PhotoSlot {
  blob: Blob
  preview: string       // URL.createObjectURL result
  uploading: boolean
  key: string | null    // R2 key after upload
  error: string | null
}
```

Returns:
```typescript
{
  inputRef: React.RefObject<HTMLInputElement>,  // for hidden file input
  slots: PhotoSlot[],
  canAdd: boolean,          // slots.length < MAX_PHOTOS
  pickPhotos: () => void,   // triggers inputRef.current.click()
  handleFiles: (e: React.ChangeEvent<HTMLInputElement>) => Promise<void>,
  removeSlot: (idx: number) => void,  // revokes blob URL, removes from array
  uploadAll: () => Promise<string[]>, // parallel upload, returns all R2 keys
  reset: () => void,        // revokes all URLs, aborts controllers, clears slots
  isUploading: boolean,     // slots.some(s => s.uploading)
}
```

Key implementation details:
- Import `compressImage` from `'../utils/imageUtils'`
- Import `useAuthStore` from `'../stores/authStore'`
- `handleFiles`: read `Array.from(e.target.files ?? [])`, reset `e.target.value = ''`, limit to `MAX_PHOTOS - slots.length` files, compress each via `compressImage`, create blob URLs via `URL.createObjectURL`
- `removeSlot(idx)`: call `URL.revokeObjectURL(prev[idx].preview)` before filtering out
- `uploadAll`: use `Promise.allSettled` for parallel upload. Each pending slot: `new FormData()`, `form.append('file', slot.blob, 'photo.jpg')`, `fetch('/api/uploads', { method:'POST', body:form, headers:{ Authorization:'Bearer ${token}' }, signal:ac.signal })`. Parse response as `{ success:boolean, data?:{ key:string } }`. On fulfilled: set `slot.key = result.key`, `uploading = false`. On rejected: set `slot.error = '업로드 실패'`. Return array of ALL keys (already-uploaded + newly-uploaded).
- `abortControllers`: `useRef<AbortController[]>([])` — NOT state
- Unmount cleanup: `useEffect(() => () => { slots.forEach(s => URL.revokeObjectURL(s.preview)); abortControllers.current.forEach(ac => ac.abort()) }, [])` — use empty dep array. To access current slots at unmount time, store URLs in a separate `useRef<string[]>` that gets updated whenever slots change.
