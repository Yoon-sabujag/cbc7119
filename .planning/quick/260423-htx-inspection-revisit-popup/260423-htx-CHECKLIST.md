## 🎯 핵심 — 지난 버그 2개 재현 테스트

### 1. 자정 후 팝업 (버그 A 해소 확인)
- [ ] 자정 지난 시점에 소화전 점검 기간 내면 이미 점검한 개소 재진입 → **(가) 팝업 뜸**
- [ ] 기록이 어제·그저께여도 오늘 활성 창이면 팝업 뜸

### 2. 오늘 계획 추가 후 팝업 (버그 B 해소 확인)
- [ ] 컴프레셔(또는 아무 카테고리) 계획을 **오늘 날짜로 추가**
- [ ] 기존 점검 완료된 개소 재진입 → **(가) 팝업 뜸** (최대 10초 내 반영, 즉시 원하면 새로고침)

---

## 📐 최종 정책 확인

| 상황 | 기대 동작 |
|------|----------|
| 오늘 활성 창 있음 + 기록 있음 (언제든) | (가) 팝업 |
| 오늘 활성 창 없음 | 팝업 없음 |
| result=bad/caution + status=open (어떤 날짜든) | **(나) 팝업 (기간 무관)** |
| CCTV/화재수신반 | 팝업 없음 |
| 접근불가 개소 | 자동 스킵 |

---

## 🔁 회귀 빠르게

- [ ] 소화전 등 오늘 활성 창인 카테고리 (가) 팝업 잘 뜸
- [ ] 활성 창 없는 카테고리에서 (가) 팝업 **안** 뜸 (과거 기록 있어도)
- [ ] (나) 주의·불량 + 미조치 개소 재진입 시 팝업 뜸 (기간 무관)
- [ ] H1 제거 유지 (이 층 점검 완료 배너 없음)
- [ ] 문구 줄바꿈 유지 / CCTV·화재수신반 제외 유지
- [ ] 콘솔에 `[revisit]` / `[revisit-fp]` 로그 **안 찍힘** (진단 로그 제거 확인)
