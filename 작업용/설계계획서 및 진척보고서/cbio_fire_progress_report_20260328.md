### 3-3. 소방 점검 페이지 ✅ 완료

**체크포인트 현황 (총 활성 개소)**

| 카테고리 | 개소 수 | 구역 분류 |
|----------|---------|-----------|
| 소화기 | 다수 (층별) | 연구동/사무동/지하 |
| 소화전 + 비상콘센트 | 다수 (층별) | 연구동/사무동/지하 |
| 특별피난계단 | 다수 | 층별 |
| 청정소화약제 | 5개소 | 연구동(7F) / 지하(B1, B4) |
| 완강기 | 13개소 | 사무동 4개 + 연구동 9개(접근불가) |
| 전실제연댐퍼 | 17개소 | 지하(B1~B5) / 지상(1F) |
| 방화셔터 | 52개소 | 연구동 북측/동측 + 사무동 + 지하 |
| DIV | 34개 측정점 | 층별 |
| 유도등 | 다수 | 층별 |
| 배연창 | 다수 | 구역별 |
| 소방용전원공급반 | 다수 | 구역별 |
| 주차장비 + 회전문 | 소수 | 고정 |
| 소방펌프 | 소수 | 고정 |

**점검 UI 플로우**
```
카테고리 선택 → 구역 선택(연구동/사무동/지하) → 층 선택 →
개소 선택(휠피커) → 결과 입력(정상/주의/불량) → 특이사항 + 사진 → 저장
```

**특수 처리**
- 완강기 접근불가 9개소: 완강기 그룹 선택 시 자동 '정상' 일괄 처리
- 전실제연댐퍼: 층 선택 → 계단전실 번호(2/3/4/5) 선택 → 기록
- 방화셔터: 연구동 내 북측/동측 구분 (description 필드)
- DIV: 별도 압력 입력 모달

**조치 플로우**
```
불량/주의 기록 → 미조치(unresolved) 상태 → 조치 후 결과 + 사진 입력 → 완료
```
