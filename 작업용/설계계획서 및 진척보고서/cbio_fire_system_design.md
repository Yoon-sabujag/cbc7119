# 2. 시스템 개요 및 목적

## 2.1 프로젝트 목적
차바이오컴플렉스 방재팀의 소방안전관리 업무를 디지털화하여 법적 점검 의무를 체계적으로 이행하고, 팀원 간 업무 효율을 높이며, 점검 기록의 신뢰성과 출력 가능한 공식 문서화를 실현하는 것을 목적으로 합니다.

## 2.2 핵심 개발 방향
- Cloudflare 유료 플랜(Workers, D1, R2, Pages) 기반으로 별도 서버 비용 없이 구축하되, 향후 방재팀 관리자의 무료 계정으로 이관 예정.
- PWA(Progressive Web App) 방식으로 모바일(iOS 16.3.1/Android 15 이상)/PC (1920x1080)공용 사용 가능
- QR코드 스캔 기반 시설물 현장 점검 지원
- 도면(SVG/이미지) 위 시설물 위치 클릭 → 점검 기록 연동
- 월별 점검 기록표 출력 인쇄 기능 내장
- 연간/월간 점검 계획표 기반 자동 일정 알림
- 승강기 법적 검사, 월점검, 고장 기록 통합 관리
- 팀원 근무표, 연차, 보수교육, 식사 이용 현황 관리
- 향후 일일보고서·소방계획서 자동 작성 기능으로 확장 예정

## 2.3 시스템 범위 (모듈 구성)

# 3. 기술 아키텍처 (Cloudflare 기반)

## 3.1 전체 아키텍처 구성
Cloudflare 유료 플랜의 서비스만으로 별도 서버 없이 전체 시스템을 구축합니다. 팀 내부 서비스이므로 트래픽이 낮아 비용 효율이 매우 높습니다.

| 레이어 | Cloudflare 서비스 | 역할 및 용도 |
|---|---|---|
| 프론트엔드 | Cloudflare Pages | React PWA 앱 호스팅. 전 세계 CDN 배포. 커스텀 도메인 지원 |
| 백엔드 API | Cloudflare Workers | REST API 서버. 인증, 데이터 처리, 비즈니스 로직 |
| 데이터베이스 | Cloudflare D1 | SQLite 기반 서버리스 DB. 점검 기록, 시설물, 사용자 저장 |
| 파일 저장 | Cloudflare R2 | 점검 사진, 도면 이미지, PDF 보고서 저장 (S3 호환) |
| 인증 | Cloudflare Access | 구글/이메일 로그인. 사내 팀원만 접근 허용 |
| 스케줄링 | Cloudflare Workers (Cron) | 점검 일정 알림 자동 발송, 월별 보고서 생성 |
| QR코드 | Workers + D1 | 시설물 QR 코드 생성·관리, 스캔 시 점검 화면 연동 |
| 푸시 알림 | Web Push API | 점검 알림, 이상 시설물 알림 팀원에게 발송 |

## 3.2 기술 스택 상세

| 구분 | 기술/도구 | 선택 이유 |
|---|---|---|
| 프론트엔드 프레임워크 | React + TypeScript | 첨부 앱과 동일 스택. 컴포넌트 재사용성 높음 |
| 스타일링 | Tailwind CSS | 빠른 UI 개발, 모바일 반응형 기본 지원 |

| PWA | Vite PWA Plugin | 오프라인 지원, 홈화면 추가, 앱 설치 경험 |
| QR 스캔 | html5-qrcode | 카메라 QR 스캔 라이브러리 (모바일 최적화) |
| QR 생성 | qrcode.js | 시설물 QR 코드 생성 및 출력 |
| PDF 출력 | react-to-print / jsPDF | 점검기록표 인쇄 및 PDF 변환 |
| 도면 연동 | SVG + React | 층별 도면 SVG 위에 시설물 마커 오버레이 |
| API 서버 | Cloudflare Workers (Hono) | 경량 웹 프레임워크, Workers 최적화 |
| DB | Cloudflare D1 (SQLite) | 서버리스 관계형 DB. 무료 한도 넉넉 |
| 파일 저장 | Cloudflare R2 | S3 호환, 무료 10GB 포함 |
| 인증 | JWT + Cloudflare Access | Cloudflare Access로 팀 접근 제한 |

# 4. 기능 상세 명세

## 4.1 인증 및 사용자 관리
**스플래시 및 로그인 화면**
- 앱 실행 시 차바이오컴플렉스 로고 + 시스템명 스플래시 표시 (2~3초)
- 로그인: 이메일 + 비밀번호 방식 (Cloudflare Access 연동 또는 자체 JWT)
- 개인별 계정: 석현민(관리자), 윤종엽(보조자), 박보융(보조자), 김병조(보조자)
- 역할별 메뉴 및 권한 차등 적용 (관리자 / 보조자)
- 로그인 후 본인 당일 근무 상태 및 오늘의 점검 일정 대시보드로 이동

## 4.2 대시보드
- 건물 전체 소방 시설물 점검 현황 요약 (정상/주의/불량/미조치/누락 건수)
- 오늘 예정 점검 항목 목록 (계획표 기반 자동 표시)
- 미조치/불량 시설물 빠른 확인 목록
- 이번 달 점검 진행률 프로그레스 바
- 승강기 이상 알림 / 최근 고장 내역 요약
- 팀 근무 현황 (오늘 주간 근무자, 당직 근무자 표시)

# 6. 개발 계획 및 로드맵

## 6.1 단계별 개발 계획 (MVP 우선)

| 단계 | 기간(예상) | 주요 개발 내용 | 핵심 산출물 | 우선순위 |
|---|---|---|---|---|
| 1단계 (MVP) | 1~2개월 | Cloudflare 환경 구축, 로그인·스플래시, 대시보드, 층별 도면 + 시설물 마커, 점검 기록 입력, QR 스캔 | 동작하는 점검 앱 | 최우선 |
| 2단계 | 2~3개월 | 점검 계획표 (연간/월간), 일정 알림, 월별 점검기록표 출력, 법적 점검 관리 | 기록 출력 가능 | 높음 |
| 3단계 | 1~2개월 | 승강기 관리 (검사/점검/고장), 근무표, 연차, 보수교육, 식사 기록 | 전체 관리 통합 | 중간 |
| 4단계 | 상시 | 일일보고서 자동 작성, 소방계획서 초안 생성, AI 어시스턴트 연동 | 보고서 자동화 | 추후 |

## 6.3 Cloudflare 비용 예측
팀 내부 서비스(4~5명 사용)이므로 Cloudflare 무료 한도 내에서 대부분 운영 가능합니다. 유료 플랜 이미 구독 중이므로 추가 비용 최소화.

| 서비스 | 무료 한도 | 예상 사용량 | 추가 비용 |
|---|---|---|---|
| Workers | 100,000 요청/일 | ~1,000 요청/일 | 무료 |
| D1 Database | 5GB / 5M 쿼리/일 | ~100MB / 수천 쿼리/일 | 무료 |
| R2 Storage | 10GB / 월 | ~1~2GB (사진+PDF) | 무료 |
| Pages | 무제한 요청 | 소량 | 무료 |
| Access | 50명 무료 | 4~5명 | 무료 |

**→ 현재 Cloudflare 유료 플랜 구독 비용 외 추가 인프라 비용 없음 (사실상 $0 추가)**

## 6.4 향후 확장 기능 (4단계)
- 일일보고서 자동 초안 생성: 당일 점검 기록 + 이상 항목 자동 취합
- 소방계획서 자동 업데이트: 변경된 시설물 현황 반영
- AI 조치 추천: 불량 항목 발생 시 과거 유사 사례 기반 조치 방법 제안
- 외부 보고: 소방청 보고 자동 양식 생성
- 모바일 오프라인 점검: 네트워크 없는 환경에서 점검 후 동기화
# 7. 보안 및 운영 정책

## 7.1 보안
- Cloudflare Access: 등록된 이메일 계정만 접근 허용 (사내 인원 외 접근 불가)
- HTTPS 강제 적용 (Cloudflare 기본 제공)
- JWT 토큰 기반 API 인증, 토큰 만료 24시간
- 역할 기반 접근 제어: 관리자(admin) / 보조자(staff) 권한 분리
- 점검 기록 삭제 불가 원칙 (수정 이력 보존)

## 7.2 데이터 백업
- Cloudflare D1 자동 백업 활성화 (일 1회)
- R2 파일은 버전 관리 활성화
- 월 1회 수동 데이터 내보내기 (CSV) 권고

## 7.3 운영 담당
- 시스템 관리자: 석현민 (소방안전관리자 대리)
- 계정 생성/삭제, QR코드 발급, 시설물 등록은 관리자만 가능
- 기술 지원 문의 발생 시 개발자와 직접 소통

## 7.4 법적 의무 이행 지원

| 법적 의무 | 시스템 지원 방식 |
|---|---|
| 소방시설 자체 점검 (월 1회) | 월별 점검 계획 → 점검 기록 → 기록표 출력 |
| 소방시설 법적 점검 (연 2회) | 법적 점검 모듈: 일정 알림, 결과 보관, 조치 추적 |
| 승강기 정기 검사 (연 1회) | 승강기 검사 모듈: 예정일 알림, 성적서 보관 |
| 승강기 월 점검 | 승강기 월점검 결과 기록 |
| 소방안전관리자 보수교육 | 보수교육 일정 등록 및 이수 기록 |
