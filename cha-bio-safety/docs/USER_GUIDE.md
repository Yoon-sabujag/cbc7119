# CHA Bio Complex 소방안전 관리 시스템 - 사용자 설명서

## 시작하기

### 앱 접속

- **모바일**: Chrome/Safari에서 https://cbc7119.pages.dev 접속 → "홈 화면에 추가"로 PWA 설치
- **데스크톱**: Chrome에서 접속 → 주소창 오른쪽 설치 아이콘 클릭 또는 와치독 프로그램이 자동 실행

### 로그인

1. 직원 카드 중 본인을 선택
2. 비밀번호 입력 (초기값: 사번 뒤 4자리)
3. "로그인" 버튼

## 대시보드

앱의 메인 화면입니다. 오늘의 업무 현황을 한눈에 파악할 수 있습니다.

### 상단 영역

- **금일 근무자**: 오늘 근무하는 직원 목록 (당직/주간/비번/휴무 표시, 연차 시 연차 유형 표시)
- **수신반 최근 이력**: 최근 48시간 이내 화재 수신반 작동 이력

### 통계 카드

| Card | Description |
|------|-------------|
| 점검 현황 | 이번 달 전체 점검 / 완료 건수 |
| 미조치 | 불량 판정 후 아직 조치하지 않은 항목 수 |
| 승강기 고장 | 현재 활성 고장 건수 |
| 연속 일수 | 연속 점검 일수 |

### 금일 일정 목록

- 오늘 예정된 점검/행사 목록
- 상태: **대기** (회색) → **진행 중** (파란색) → **완료** (녹색)
- 각 항목 클릭 → 상태 변경 가능

### 월간 진행률

- 도넛 차트로 이번 달 점검 진행 상태 표시
- 30초마다 자동 새로고침

## 소방시설 점검

### 일반 점검 (점검 탭)

현장에서 소방시설을 점검하고 결과를 기록하는 핵심 기능입니다.

1. **구역 선택**: 연구동 / 사무동 / 지하
2. **층 선택**: 해당 구역의 층
3. **점검 개소 선택**: 휠 피커로 개소 선택
4. **카테고리**: 소화기, 유도등, 감지기 등 15개 카테고리
5. **점검 결과 입력**:
   - **정상** (녹색) — 이상 없음
   - **주의** (노란색) — 경미한 이상
   - **불량** (빨간색) — 즉시 조치 필요
6. **특이사항**: 메모 입력 (선택)
7. **사진 촬영**: 카메라로 현장 사진 첨부 (선택)
8. **점검 기록 저장** 버튼

### QR 스캔 점검 (QR 탭)

1. QR 스캔 화면에서 카메라로 개소 QR 코드 스캔
2. 자동으로 해당 개소 정보 로드
3. 점검 결과/메모/사진 입력 → 저장

### 조치 관리 (조치 탭)

불량/주의로 판정된 항목의 후속 조치를 관리합니다.

**목록 화면:**
- 미조치 / 완료 탭으로 필터링
- 각 항목: 위치, 카테고리, 판정결과, 일시 표시
- 조치보고서 다운로드 (HTML 형식)

**상세 화면:**
1. 점검 정보 확인 (카테고리, 위치, 일시, 결과)
2. **조치 내용** 입력:
   - 메모 (어떤 조치를 했는지)
   - 자재 선택 (유도등: 본체/예비전원 등)
   - 자재 수량
   - 조치 사진 촬영
3. **조치 완료** 버튼 → 완료 처리
4. **조치 취소** — 이미 완료된 조치를 되돌림 (관리자)
5. **기록 삭제** — 점검 기록 자체 삭제 (관리자)

## 건물 도면

건물 평면도 위에 소방설비 위치를 마커로 표시하고, 마커에서 바로 점검할 수 있습니다.

### 도면 유형 (4종)

| Tab | Markers |
|-----|---------|
| 유도등 | 천장출구, 벽면출구, 거실/복도/계단/객석 통로 |
| 감지기 | 연기감지기, 열감지기 |
| 스프링클러 | 폐쇄형, 개방형, 킹형, 시험밸브 |
| 소화기·소화전 | 분말3.3kg, 분말20kg, 할로겐, K급주방, 소화전, 완강기, DIV |

### 마커 편집

1. 우상단 **마커 편집** 버튼
2. 도면 위 빈 곳을 **꾹 누르기** (모바일) 또는 **더블클릭** (데스크톱) → 추가 모달
3. 마커 종류 선택
4. **점검 개소 연결**: 기존 개소 선택 또는 **새로 등록** (소화기 한정)
5. **추가** 버튼

**소화기 새로 등록:**
- 분말/할로겐/K급 마커 선택 시 "기존 개소 연결 / 새로 등록" 토글
- 위치, 제조업체, 제조년월, 형식승인, 접두문자, 증지번호, 제조번호 입력
- check_points + extinguishers 테이블에 자동 생성

### 마커에서 점검

1. 마커 클릭 → 하단 정보 패널
2. **점검 기록 입력** 버튼
3. 소화기인 경우 상세정보 표시 (위치, 제조업체, 형식승인, 증지번호 등)
4. 결과 선택 (정상/주의/불량) + 메모 + 사진 → 저장

## 승강기 관리

### 5개 탭

**1. 호기 목록**
- 전체 승강기 카드 (번호, 상태, 활성고장 수)
- 상태: 정상 / 고장 / 정비 / 사용중지

**2. 고장 이력**
- 고장 접수 (일시, 위치, 증상, 사진)
- 고장 처리 (원인, 조치내용, 완료일시)

**3. 수리 이력**
- 수리 기록 등록 (업체, 내용, 비용, 사진)

**4. 점검 이력**
- 정기 점검 기록 (월/분기/반기/연)
- 점검 결과 첨부

**5. 연간 점검**
- 검사기관 점검 일정
- 인증서 업로드/다운로드
- 지적사항 등록 및 조치

---

## DIV (유수검지장치) 관리

### 3개 탭

**1. 압력 트렌드**
- 34개 측정점 (8-1층~B5층)
- 각 측정점: 1차압, 2차압, 챔버압
- 상태 표시: 정상(녹색) / 경고(노란색, ±10%) / 위험(빨간색, ±20%)
- 연도 필터

**2. 배수 주기**
- 배수 작업 기록 등록/조회
- DIV별 필터링

**3. 오일 주기**
- 컴프레셔 오일 교체/보충 기록

## 일정 관리

### 월간 점검 일정 (/schedule)

1. 월/년 선택
2. **자동 생성** 버튼 → 19개 카테고리 기반 월간 일정 자동 배치
3. 일정 항목 클릭 → 상세/수정/삭제

   - 날짜 선택

   - 메모 (점검 가이드라인)

- 월간 직원×일 매트릭스
- 색상별 근무유형 (당/비/주/휴)
- 공휴일 빨간색 표시
- **엑셀 다운로드** 버튼

## 문서 출력

### 소방설비 점검일지 (/reports)

10종의 점검일지를 엑셀로 생성합니다.

1. 점검일지 종류 선택 (카드)
2. 연도 선택 (월별 일지는 월도 선택)
3. **미리보기** — 데이터 확인
4. **다운로드** — 엑셀 파일 생성

### 일별업무일지 (/daily-report)

1. 날짜 이동 (← →)
2. 금일/명일 업무내용 입력
3. 소방/피난/가스/기타 점검결과 입력
4. 자동 저장 (디바운스)
5. **엑셀 다운로드** — 월간 업무일지

### 업무수행기록표 (/worklog)

1. 월 선택
2. 관리자명, 점검 내용/결과/조치 입력
3. 보고 방법/조치 방법 선택
4. **저장** → **엑셀 다운로드**

### 문서 보관소 (/documents)

- **소방계획서** 탭: 연도별 소방계획서 업로드/다운로드/삭제
- **소방훈련자료** 탭: 훈련 자료 업로드/다운로드/삭제
- 대용량 파일 지원 (멀티파트 업로드)

- 비밀번호 초기화 (사번 뒤 4자리로 리셋)

- 직원별 교육 이수 카드

## 설정

화면 우측 상단 톱니바퀴(설정) 아이콘을 눌러 접근합니다.

### 알림 설정

푸시 알림을 활성화하면 아래 알림을 받을 수 있습니다:
- 금일 점검 일정 (08:45)
- 전일 미완료 점검 (08:45)
- 미조치 항목 (08:45)
- 행사 15분 전 / 5분 전
- 교육 D-30

**주의:** 각 직원이 자기 기기에서 직접 구독해야 합니다. 비번/휴무/연차인 날에는 알림이 발송되지 않습니다.

### 메뉴 설정

사이드 메뉴 항목의 순서와 표시 여부를 커스터마이즈할 수 있습니다.

### 계정

- 연락처/이메일 수정
- 비밀번호 변경 (4자 이상)

### 데이터베이스 (관리자 전용)

**DB 백업/복원:**
- **백업**: 전체 데이터베이스를 .sql 파일로 다운로드
- **업로드**: .sql 파일로 데이터베이스 복원 (기존 데이터 덮어쓰기)

**파일 백업/복원:**
- **백업**: 점검 사진 등을 .zip으로 다운로드 (소방계획서·훈련자료·미리보기 제외)
- **업로드**: .zip 파일에서 R2 스토리지로 복원

### 캐시 초기화

앱이 이상하게 동작할 때 서비스워커 캐시를 초기화합니다. 로그인 상태는 유지됩니다.

## 데스크톱 와치독 (Windows)

### 설치

1. `CHA Bio File Organizer.bat` 실행
2. 최초 실행 시 설정 화면:
   - 루트 폴더 선택 (파일이 정리될 기본 폴더)
   - 앱 자동 열기 여부
3. 시스템 트레이에 상주

### 기능

- **파일 자동 분류**: 다운로드 폴더에 점검일지, 근무표, QR PDF 등이 생기면 자동으로 지정 폴더로 이동
  - 예: `2026년도_DIV점검표_월초.xlsx` → `4. 소방설비점검일지/유수검지장치(월초)/2026년/`
- **백업 파일 자동 이동**: DB 백업(.sql), 파일 백업(.zip) 자동 분류
- **Chrome 백그라운드 실행**: 앱 창을 닫아도 푸시 알림 수신 가능하도록 자동 설정
- **PWA 자동 실행**: 시작 시 앱 자동 열기

## FAQ

**Q: 점검 기록을 실수로 잘못 입력했어요.**
A: 같은 개소에 다시 점검 기록을 입력하면 최신 기록으로 덮어씌워집니다.

**Q: 불량 판정을 정상으로 바꾸고 싶어요.**
A: 조치 페이지에서 해당 항목을 조치 완료 처리하거나, 관리자가 기록을 삭제할 수 있습니다.

**Q: 푸시 알림이 안 와요.**
A: 설정 > 알림에서 푸시 알림이 켜져있는지 확인하세요. 브라우저 알림 권한도 "허용됨"이어야 합니다. 각 기기에서 직접 구독해야 합니다.

**Q: 비밀번호를 잊었어요.**
A: 관리자에게 요청하면 사번 뒤 4자리로 초기화해줄 수 있습니다. (직원 관리 페이지)

**Q: 데이터 백업은 얼마나 자주 해야 하나요?**
A: 주 1회 권장. 설정 > 데이터베이스에서 DB 백업 + 파일 백업을 모두 받아두세요.

**Q: 소화기를 새로 비치했는데 앱에 어떻게 추가하나요?**
A: 도면 페이지 > 소화기·소화전 탭 > 마커 편집 > 해당 위치 꾹 누르기 > 소화기 종류 선택 > "새로 등록" > 정보 입력 > 추가

**Q: 엑셀 파일이 다운로드되면 어디로 가나요?**
A: 다운로드 폴더에 저장됩니다. 데스크톱에서 와치독이 실행 중이면 자동으로 지정 폴더로 이동합니다.
